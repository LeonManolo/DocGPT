DocGPT generates comprehensive doc comments for your code leveraging the Chat GPT API. This IntelliJ plugin streamlines your documentation workflow and ensures a consistent comment style throughout your codebase.

Currently, DocGPT supports Java, Kotlin, and Dart languages. To make use of this plugin, you need to add your API key in the IntelliJ settings under Settings/Tools/DocGPT. To generate a doc comment for a function, select the function's name with the cursor, press Alt + Enter, then choose "Generate doc comment" from the menu.

Please note that DocGPT currently supports generating doc comments for functions only. The plugin is still under development and has not been extensively tested yet. As such, it is recommended to use it with caution and understanding of these limitations.

For further details and configuration options, please visit the settings of the plugin.

To navigate to related settings directly from this description, add a link with `settings://$` optionally followed by `?$` to pre-select a UI element.
